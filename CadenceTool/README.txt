Cadence Tool - SVN 插件安装程序
====================================

版本: 1.2
日期: 2025-12-04
作者: Sloan Chi

简介:
------------
为 Cadence Capture CIS 17.2 提供 SVN 版本控制集成的浮动工具栏插件。

主要功能:
---------
✓ SVN 更新 (Update) - 快速更新设计文件
✓ SVN 提交 (Commit) - 提交设计变更
✓ SVN 清理 (Cleanup) - 清理 SVN 工作区
✓ SVN 历史 (Show Log) - 查看版本历史记录
✓ 浮动工具栏 - 轻量级界面，位于屏幕右下角
✓ 快捷键支持 - 提高工作效率

安装步骤:
------
1. 双击运行 "Cadence Tool.exe"
2. 选择操作选项:
   [1] Install Plugin   - 安装插件
   [2] Uninstall Plugin - 卸载插件
   [3] Exit            - 退出程序
3. 程序会自动检测 Cadence 安装路径
4. 按提示完成安装
5. 重启 Cadence Capture 生效

使用说明:
------
启动 Cadence Capture 后，工具栏自动出现在屏幕右下角。

【工具栏按钮】
• U (绿色) - SVN Update    更新当前设计
• C (蓝色) - SVN Commit    提交当前设计
• L (橙色) - SVN Cleanup   清理工作区
• S (紫色) - SVN Show Log  查看历史记录
• A (灰色) - About Plugin  关于插件

【快捷键】
• Ctrl+U - SVN 更新
• Ctrl+M - SVN 提交
• Ctrl+H - SVN 历史记录

【菜单访问】
Tools > SVN Tools > 选择对应功能

【鼠标悬停提示】
将鼠标移动到按钮上，窗口标题栏会显示该按钮的功能说明。

系统要求:
-------------
• Cadence Capture CIS 17.2 (SPB 17.2-2016)
• TortoiseSVN 1.9 或更高版本
• Windows 7/10/11

常见问题:
--------
Q: 工具栏没有显示怎么办？
A: 在 Cadence 中选择菜单 Tools > SVN Tools > Show Toolbar

Q: 如何卸载插件？
A: 重新运行 "Cadence Tool.exe"，选择 [2] Uninstall Plugin

Q: SVN 命令不工作？
A: 确保 TortoiseSVN 已安装在默认路径：
   C:\Program Files\TortoiseSVN\bin\TortoiseProc.exe

Q: 工具栏位置能否调整？
A: 工具栏会自动定位在屏幕右下角任务栏上方，可以手动拖动。

Description:
------------
A floating toolbar plugin providing SVN version control integration 
for Cadence Capture CIS 17.2.

Features:
---------
✓ SVN Update - Quick update design files
✓ SVN Commit - Commit design changes
✓ SVN Cleanup - Clean SVN working copy
✓ SVN Show Log - View version history
✓ Floating toolbar - Lightweight UI at bottom-right corner
✓ Keyboard shortcuts - Improve work efficiency

Installation:
------
1. Double-click "Cadence Tool.exe"
2. Select option:
   [1] Install Plugin
   [2] Uninstall Plugin
   [3] Exit
3. Auto-detect Cadence installation path
4. Follow on-screen instructions
5. Restart Cadence Capture to take effect

Usage:
------
Toolbar appears automatically at bottom-right corner after Cadence starts.

【Toolbar Buttons】
• U (Green)  - SVN Update
• C (Blue)   - SVN Commit
• L (Orange) - SVN Cleanup
• S (Purple) - SVN Show Log
• A (Gray)   - About Plugin

【Keyboard Shortcuts】
• Ctrl+U - SVN Update
• Ctrl+M - SVN Commit
• Ctrl+H - SVN Show Log

【Menu Access】
Tools > SVN Tools > Select function

【Mouse Hover】
Move mouse over button to see function description in title bar.

-------------

FAQ:
----
Q: Toolbar not showing?
A: Select menu: Tools > SVN Tools > Show Toolbar

Q: How to uninstall?
A: Run "Cadence Tool.exe" again, select [2] Uninstall Plugin

Q: SVN commands not working?
A: Ensure TortoiseSVN installed at default path:
   C:\Program Files\TortoiseSVN\bin\TortoiseProc.exe
